Constraint Class: Branch Transition & Propagation Split

When a repository changes its default branch (e.g., master to main), the multi-layer CDN architecture can completely shatter, serving three entirely different states simultaneously. This is a powerful diagnostic tool for testing propagation speeds.

Observer-Bounded State Matrix (GPT-5.4)

Diagnostic Finding: Pages has caught up to the raw main branch, but the raw master endpoint is still serving the older, pre-transition cached state.
